<hr>
 
&emsp;&emsp;对iOS设备（iOS8之前及iOS8和iOS10新特性）实现本地推送的封裝，方便统一管理及使用，包括设备授权、多种样式的本地推送和前台接收本地推送的回调函数等多种特性，主要功能如下:
 - 申请设备推送权限
 - 普通样式推送
 - 图片样式推送
 - 视频样式推送
 - 定时推送
 - 定点推送 
 - 交互推送
 - 前台接收推送的回调函数
